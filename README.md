# GitCompletion

- copy the both files to your home directory ```~/```
- add the folowing code to your ```.bashrc``` or ```.bash_profile```

```
if [ -f ~/.git-completion.bash ]; then
  source ~/.git-completion.bash
fi
if [ -f ~/.git-prompt.sh ]; then
  source ~/.git-prompt.sh
fi
```

- open a new console
